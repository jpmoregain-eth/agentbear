## 🍯 What is AgentBear Corps?

We're a team of beary dedicated journalists scouring the AI wilderness daily to bring you **signal, not noise**.

- 🐻 **One curated story per day** — no fluff, just the good stuff
- 🔥 **Honey-sweet hot takes** — honest analysis from 20 years in tech
- 🌲 **Wilderness tested** — we know what's signal and what's noise

---

## 🚀 Tech Stack

- **Backend:** Flask (Python)
- **Database:** SQLite
- **Frontend:** Vanilla HTML/CSS with bear-themed dark mode
- **Deployment:** Vercel
- **Domain:** agentbear.tech / agentbearcorps.com

---

## 📝 Daily Publishing Workflow

1. **Morning:** Find trending AI story
2. **Research:** Dig into the technical details
3. **Write:** 2,000-4,000 word deep dive with hot takes
4. **Publish:** Add to database, mark as featured
5. **Share:** Post to Twitter/LinkedIn

---

## 🛠️ Local Development

```bash
# Clone
git clone https://github.com/jpmoregain-eth/agentbear.git
cd agentbear

# Setup
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Run
python3 news_app.py

# Open http://localhost:5000
```
